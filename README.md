# EEG Seizure Classification

Binary seizure detection from EEG signals using scalogram-based CNN and ResNet18 models.

## Repository contents

- `README.md` — this file
- `requirements.txt` — Python dependencies
- `data.csv` — raw EEG data
- `scalograms/` — pre-computed scalogram arrays (generated by notebook 2)
- `plots/` — saved output figures
- `best_baseline_cnn.pt`, `best_baseline_cnn_recall.pt` — trained Baseline CNN checkpoints
- `best_resnet18.pt`, `best_resnet18_recall.pt` — trained ResNet18 checkpoints
- notebooks — see Running the notebooks section below

---

## Requirements

- Python 3.10 or newer
- An NVIDIA GPU with CUDA support is strongly recommended (training on CPU will be very slow)

---

## Installation

1. Clone the repository and enter the project folder:

```
git clone <repo-url>
cd EEG_classification
```

2. Create and activate a virtual environment:

```
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

3. Install dependencies:

```
pip install -r requirements.txt
```

For GPU support, replace the torch install with the CUDA build. Check your CUDA version with `nvidia-smi`, then run:

```
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
```

4. Launch Jupyter:

```
jupyter notebook
```

---

## Running the notebooks

Run the notebooks in order. Each step depends on the output of the previous one.

1. `01_data_preparation.ipynb` — loads and explores data.csv, splits into train / val / test
2. `02_scalograms.ipynb` — converts EEG signals to scalogram images, saves .npy files to scalograms/
3. `03_baseline_cnn_F1.ipynb` — trains Baseline CNN with Optuna hyperparameter search, optimised for F1
4. `04_baseline_cnn_recall.ipynb` — same as above, optimised for Recall
5. `05_resnet18_F1.ipynb` — trains ResNet18 with Optuna hyperparameter search, optimised for F1
6. `06_resnet18_Recall.ipynb` — same as above, optimised for Recall
7. `07_confusion_matrices.ipynb` — loads all 4 checkpoints and produces a 2x2 confusion matrix figure

Notebooks 3-6 each run 30 Optuna trials x 5-fold CV and can take around 1 hour per notebook on an RTX 3090. On CPU, expect significantly longer runtimes.
